# 💉 승압제 주입속도 계산기 (Vasopressor Infusion Rate Calculator)

체중기반 용량(`mcg/kg/min`)과 펌프 주입속도(`cc/hr`)를 **양방향**으로 변환해 주는 단일 페이지 웹 도구입니다.
설치 없이 브라우저만 있으면 바로 사용할 수 있습니다.

🔗 **바로 사용하기:** https://jin02.github.io/vasopressor-calculator/

---

## ✨ 주요 기능

- **약물 프리셋 5종** — 노르에피네프린(노르핀)·에피네프린·도파민·도부타민·페닐레프린
  선택하면 표준 희석값이 자동으로 채워집니다.
- **양방향 변환**
  - `용량 → 주입속도` : `mcg/kg/min` → **`cc/hr`**
  - `주입속도 → 용량` : `cc/hr` → **`mcg/kg/min`**
- **농도 자동 계산** — 약물량(mg)과 희석볼륨(mL)으로 `mcg/mL` 자동 산출
- **권장 용량범위 경고** — 범위 막대 + 마커로 시각화하고, 하한 미만·범위 내·상한 초과를 색상으로 알림

---

## 📖 사용 방법

1. **약물 선택**
   상단에서 약물을 누르면 표준 희석값(약물량·볼륨)이 자동으로 입력됩니다.

2. **희석 / 환자 정보 입력**
   | 입력 | 설명 |
   |------|------|
   | 약물 총량 (mg) | 백에 섞은 약물의 총량 |
   | 희석 후 총 볼륨 (mL) | 희석액을 포함한 최종 볼륨 |
   | 환자 체중 (kg) | 환자 체중 |
   | 농도 (mcg/mL) | **자동 계산** (입력 불필요) |

3. **계산 방향 선택**
   - **용량 → 주입속도** : 목표 용량을 넣으면 펌프 속도(cc/hr)가 나옵니다.
   - **주입속도 → 용량** : 현재 펌프 속도를 넣으면 실제 용량(mcg/kg/min)이 나옵니다.

4. **결과 확인**
   큰 숫자로 결과가 표시되고, 아래 막대에서 권장 범위 대비 현재 위치를 확인할 수 있습니다.
   - ✅ 초록 : 권장 치료범위 내
   - ⚠️ 노랑 : 권장 하한 미만
   - 🚨 빨강 : 권장 상한 초과 (재확인 필요)

## 🧮 계산 공식

**농도**

```
농도 C (mcg/mL) = 약물량(mg) × 1000 ÷ 희석볼륨(mL)
```

**용량 → 주입속도**

```
주입속도 R (mL/hr) = 용량 D(mcg/kg/min) × 체중(kg) × 60 ÷ 농도 C(mcg/mL)
```

**주입속도 → 용량**

```
용량 D (mcg/kg/min) = 주입속도 R(mL/hr) × 농도 C(mcg/mL) ÷ ( 체중(kg) × 60 )
```

> 예시: 노르에피네프린 4 mg / 250 mL → 농도 16 mcg/mL,
> 체중 60 kg, 용량 0.1 mcg/kg/min → `0.1 × 60 × 60 ÷ 16 = 22.5 cc/hr`
